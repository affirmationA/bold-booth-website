### Why This Works:

1. **Variable Rewards** 
   - Every interaction creates a unique effect

   - Click = confetti + ripple (double reward)

   - Confetti = celebration = weddings

   - Auto-pulsing creates "what's that?" curiosity
   - Staggered animations create rhythm
   - Scroll reveals create "what's next?" excitement
   - Parallax adds depth and dimension

5. **Memory Formation**
   - Unique experience = memorable
   - Emotional + visual = strong memory
   - "That website with the confetti!"
   - Increases brand recall
